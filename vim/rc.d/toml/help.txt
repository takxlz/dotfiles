# hook_add = 'imap <C-L> <Plug>CapsLockToggle'
deinによってプラグインが読み込まれたときに発火される設定
プラグインの設定や，マッピングを書く

# hook_source = 'let g:echodoc_enable_at_startup = 1'
プラグインを読み込む直前に実行される
hook_addと違い，プラグインを読み込むまでは実行されない

など



----- 遅延読み込み条件（dein_lazy） -----
# on_ft = ['foo']
.fooファイルタイプで読み込む

# on_i = 1
インサートモードに入ったら読み込む

# on_source = ['foo.vim']
foo.vimを読み込んだとき一緒に読み込む

# on_cmd = ['foo']
fooコマンドを打ったら読み込む

# gui = 1
gvimのときだけ読み込む
