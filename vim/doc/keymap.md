※ カスタムキーバインド・・・_(アンダースコア)で囲む  
※ 基本的に「{count}operator motion」のように組み合わせることができる

### mode
| key               | description                                    | remark |
|-------------------|------------------------------------------------|--------|
| _jj_, esc         | ノーマルモードに入る                           |        |
| i                 | カーソル左からインサートモードに入る           |        |
| a                 | カーソル右からインサートモードに入る           |        |
| I                 | 行頭からインサートモードに入る                 |        |
| A                 | 行末からインサートモードに入る                 |        |
| o                 | 下に行を追加してインサートモードに入る         |        |
| O                 | 上に行を追加してインサートモードに入る         |        |
| s                 | カーソルの文字を削除してインサートモードに入る |        |
| v                 | ヴィジュアルモードに入る                       |        |
| V                 | 行単位のヴィジュアルモードに入る               |        |
| C-v               | 短形選択のヴィジュアルモードに入る             |        |
|-------------------|------------------------------------------------|--------|
| {count}i{char}esc | {char}を{count}の数だけ入力する                |        |
|-------------------|------------------------------------------------|--------|


### move (motion)
| key       | description                      | remark  |
|-----------|----------------------------------|---------|
| h         | ← 1文字                         | {count} |
| b (B)     | ← 単語の先頭 (記号無視)         | {count} |
| ge (gE)   | ← 前の単語の末尾 (記号無視)     |         |
| T{char}   | ← 行内の{char}の前へ移動        |         |
| F{char}   | ← 行内の{char}へ移動            |         |
| ^ (0)     | ← 行頭 (インデント、SPCを無視)  |         |
|-----------|----------------------------------|---------|
| l         | → 1文字                         | {count} |
| e (E)     | → 単語の末尾 (記号無視)         | {count} |
| w (W)     | → 次の単語の先頭 (記号無視)     | {count} |
| t{char}   | → 行内の{char}の前へ移動        |         |
| f{char}   | → 行内の{char}へ移動            |         |
| $         | → 行末                          |         |
|-----------|----------------------------------|---------|
| k (gk)    | ↑ 1文字 (表示単位)              | {count} |
| -         | ↑ 上の行の先頭                  |         |
| {         | ↑ 上の段落の先頭                |         |
| H         | ↑ 画面の上端行                  | {count} |
| C-b (C-u) | ↑ 1ページスクロール (1/2ページ) |         |
| gg        | ↑ 最初の行                      |         |
|-----------|----------------------------------|---------|
| j (gj)    | ↓ 1文字 (表示単位)              | {count} |
| +         | ↓ 下の行の先頭                  |         |
| }         | ↓ 下の段落の先頭                |         |
| L         | ↓ 画面の下端行                  | {count} |
| C-f (C-d) | ↓ 1ページスクロール (1/2ページ) |         |
| G         | ↓ 最後の行                      |         |
|-----------|----------------------------------|---------|
| :{num}    | {num}行                          |         |
| %         | 対応する括弧                     |         |
| g;        | 変更してきた箇所に戻る           |         |
| g,        | 変更してきた箇所に進む           |         |
| M         | 画面の中央行                     |         |
| gm        | 画面幅の中央                     |         |
| z<ret>    | 現在行を画面のトップにする       |         |
| zz        | 現在行を画面の中央にする         |         |
|-----------|----------------------------------|---------|


### yank (y: operator)
| key     | description          | remark  |
|---------|----------------------|---------|
| y[l, h] | [l, h]の方向の1文字  | {count} |
| y[j, k] | [j, k]の方向の1行    | {count} |
| yw      | 1単語                | {count} |
| yy      | 1行                  | {count} |
| y0      | 行頭からカーソル直前 | {count} |
| y$      | カーソルから行末     | {count} |
| y       | 選択範囲             |         |
|---------|----------------------|---------|


### paste (p: operator)
| key        | description                       | remark              |
|------------|-----------------------------------|---------------------|
| p (P)      | 「無名レジストリ」カーソル右 (左) | {count}             |
| p (P)      | 「ヤンクレジストリ」下の行 (下)   | yyの後のみ, {count} |
| _C-p_, "0p | 「ヤンクレジストリ」              | {count}             |
| "+p        | 「クリップボード」                | {count}             |
|------------|-----------------------------------|---------------------|
※ x,sで削除した文字は「無名レジストリ」に格納される(設定で無効)


### delete (d: operator)
| key     | description                  | remark  |
|---------|------------------------------|---------|
| d[l, h] | [l, h]の方向の1文字          | {count} |
| d[j, k] | [j, k]の方向の1行            | {count} |
| dw      | カーソル位置から1単語        | {count} |
| diw     | カーソル位置にある1単語      | {count} |
| dd      | 1行                          | {count} |
| d0      | 行頭からカーソル直前         | {count} |
| d$      | カーソル位置から行末         | {count} |
|---------|------------------------------|---------|
| ddp     | カーソル行と下の行の入れ替え | {count} |


### change (c: operator)
| key      | description                                            | remark  |
|----------|--------------------------------------------------------|---------|
| c[l, h]  | [l, h]の方向の1文字                                    | {count} |
| c[j, k]  | [j, k]の方向の1行                                      | {count} |
| cw       | カーソル位置から単語を消してインサートモード           |         |
| ciw      | カーソル位置にある単語を消してインサートモード         |         |
| cc       | 1行                                                    | {count} |
| c0       | 行頭からカーソル直前                                   | {count} |
| c$       | カーソル位置から行末                                   | {count} |
| ci{char} | {char}で囲まれた中身を消してインサートモード           |         |
| ca{char} | {char}で囲まれた中身を{char}ごと消してインサートモード |         |
| cit      | HTMLタグに囲まれた中身を消してインサートモード         |         |
|----------|--------------------------------------------------------|---------|
