# Neovim チートシート

leader = Space

---

## キーマップ方針

### プレフィックスの分類

| プレフィックス | 用途                 | 覚え方       |
| -------------- | -------------------- | ------------ |
| `s` + キー     | ウィンドウ・タブ操作 | split/screen |
| `<leader>f`    | 検索                 | find         |
| `<leader>g`    | git操作              | git          |
| `<leader>h`    | hunk操作             | hunk         |
| `<leader>c`    | コードアクション     | code         |
| `g` + キー     | LSPジャンプ          | goto         |
| `[` / `]`      | 前/次へジャンプ      | 方向         |

### 単独キー

| キー        | 用途                   | 覚え方                   |
| ----------- | ---------------------- | ------------------------ |
| `-`         | oil.nvim（ファイラー） | 親ディレクトリへ戻る慣例 |
| `<leader>e` | neo-tree（ツリー表示） | explorer                 |
| `<leader>a` | aerial（シンボル一覧） | aerial                   |
| `<leader>w` | 保存                   | write                    |
| `<leader>?` | チートシート表示       | help                     |

### `[` / `]` ジャンプの統一パターン

| サフィックス | 対象               |
| ------------ | ------------------ |
| `b`          | バッファ           |
| `d`          | 診断（diagnostic） |
| `h`          | hunk（git変更）    |
| `t`          | TODOコメント       |

## 基本操作

| キー         | モード | 機能                                |
| ------------ | ------ | ----------------------------------- |
| `jj`         | Insert | ノーマルモードに戻る                |
| `<Esc>`      | Normal | 検索ハイライト + 単語ハイライト解除 |
| `<Esc><Esc>` | Normal | フロートウィンドウを閉じる          |
| `<leader>w`  | Normal | 保存                                |
| `<leader>?`  | Normal | チートシート表示（フロート）        |

## ウィンドウ操作 (s + キー)

| キー                      | 機能                          |
| ------------------------- | ----------------------------- |
| `sv`                      | 縦分割                        |
| `ss`                      | 横分割                        |
| `sh` / `sj` / `sk` / `sl` | ウィンドウ移動（左/下/上/右） |
| `sq`                      | ウィンドウを閉じる            |
| `s<Space>`                | リサイズモード（hydra）       |

### リサイズモード中

| キー      | 機能               |
| --------- | ------------------ |
| `>` / `<` | 幅を広く / 狭く    |
| `+` / `-` | 高さを高く / 低く  |
| `=`       | 均等にする         |
| `<Esc>`   | リサイズモード終了 |

## タブ操作

| キー | 機能       |
| ---- | ---------- |
| `st` | 新しいタブ |
| `sn` | 次のタブ   |
| `sp` | 前のタブ   |

## バッファ操作

| キー | 機能         |
| ---- | ------------ |
| `[b` | 前のバッファ |
| `]b` | 次のバッファ |

## テキスト編集

| キー        | モード        | 機能                         |
| ----------- | ------------- | ---------------------------- |
| `J`         | Visual        | 選択行を下に移動             |
| `K`         | Visual        | 選択行を上に移動             |
| `<leader>p` | Visual        | ヤンクを上書きしない貼り付け |
| `gc`        | Visual        | 選択範囲のコメントトグル     |
| `gcc`       | Normal        | 行コメントトグル             |
| `sa`        | Normal/Visual | 囲みを追加（mini.surround）  |
| `sd`        | Normal        | 囲みを削除（mini.surround）  |
| `sr`        | Normal        | 囲みを置換（mini.surround）  |

## 単語ハイライト

| キー              | 機能                                           |
| ----------------- | ---------------------------------------------- |
| `<leader><Space>` | カーソル下の単語をハイライト（複数可、色分け） |
| `<Esc>`           | すべてのハイライトをクリア                     |

## Git - gitsigns

| キー         | キーマップ            |
| ------------ | --------------------- |
| `]h` / `[h`  | 次/前のhunkへジャンプ |
| `<leader>hs` | hunkをステージ        |
| `<leader>hr` | hunkをリセット        |
| `<leader>hp` | hunkをプレビュー      |
| `<leader>hb` | 行のblame表示         |

| コマンド                              | 機能                  |
| ------------------------------------- | --------------------- |
| `:Gitsigns toggle_signs`              | 変更マークの表示切替  |
| `:Gitsigns toggle_current_line_blame` | 行blameの常時表示切替 |

## Git - diffview

| キー         | キーマップ               |
| ------------ | ------------------------ |
| `<leader>gv` | 変更ファイル一覧（diff） |
| `<leader>gh` | 現在ファイルの履歴       |

| コマンド                 | 機能                   |
| ------------------------ | ---------------------- |
| `:DiffviewOpen`          | 変更ファイル一覧を開く |
| `:DiffviewFileHistory %` | 現在ファイルの履歴     |
| `:DiffviewFileHistory`   | リポジトリ全体の履歴   |
| `:DiffviewClose`         | diffviewを閉じる       |

## Git - fugitive

| キー         | キーマップ         |
| ------------ | ------------------ |
| `<leader>gs` | git status         |
| `<leader>gd` | git diff（縦分割） |
| `<leader>gb` | git blame          |
| `<leader>gl` | git log            |

| コマンド             | 機能                                             |
| -------------------- | ------------------------------------------------ |
| `:Git`               | git status（ステージ/コミット操作）              |
| `:Git commit`        | コミット                                         |
| `:Git push`          | プッシュ                                         |
| `:Git pull`          | プル                                             |
| `:Git blame`         | blame表示                                        |
| `:Git log`           | ログ表示                                         |
| `:Git log --oneline` | ログ表示（1行）                                  |
| `:Gvdiffsplit`       | diff（縦分割）                                   |
| `:Gdiffsplit`        | diff（横分割） / コンフリクト解消（3-way merge） |
| `:diffget`           | diff中にもう一方の変更を取り込む                 |
| `:diffput`           | diff中に自分の変更を相手に送る                   |

## 補完 (blink.cmp)

### インサートモード（コード補完）

| キー      | 機能     |
| --------- | -------- |
| `<Tab>`   | 次の候補 |
| `<S-Tab>` | 前の候補 |
| `<Down>`  | 次の候補 |
| `<Up>`    | 前の候補 |
| `<CR>`    | 補完確定 |

### コマンドラインモード（`:`, `/`, `?`）

| キー      | 機能                               |
| --------- | ---------------------------------- |
| `<Tab>`   | 次の候補                           |
| `<S-Tab>` | 前の候補                           |
| `<Down>`  | 次の候補                           |
| `<Up>`    | 前の候補                           |
| `<CR>`    | 補完確定（未選択時はコマンド実行） |

※ コマンドライン入力時は候補が自動表示される

| コマンド                 | 機能                |
| ------------------------ | ------------------- |
| `:checkhealth blink.cmp` | blink.cmpの動作確認 |
